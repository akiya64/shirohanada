== Description ==
Shirohanada is japanese ancient times color name, like skyblue. This is minimal, flat, responsive WordPress theme based Shirohanada color.

* Responsive Layout
* Custom Menu
* Custom Header
* Front Page
* Social Links

== Installation ==

Manual instrallation

1. Upload the all theme files '/themes/shirohanada' directory.
2. Activate the theme throgh the 'Theme' screen in WordPress.
3. Customize menu in frontpage. 'Theme Location' checked 'Front Page Menu' in 'Menu' screen.
